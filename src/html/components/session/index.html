после сессии по программе

После такой сесси коуч выводит вас из состояния гипнотического транса.

Вы “просыпаетесь” меняется ваше состояние, внешне это может проявлятся хорошим настроением, исчезновением волнениея тревоги и переживаний.

Освобождается энергия и блоки снимаются. Увеличивается и улучшается ваше ресурсное состояние.
